# LinkedIn Tech Agent

## Project Overview

LinkedIn Tech Agent is a personal AI engineering project that aims to automate the creation of high-quality, technology-focused LinkedIn posts.

The primary goal of this project is not only to build an AI agent, but also to understand AI engineering fundamentals from first principles.

## Current Status

- Environment setup completed
- Python development environment configured
- Git repository initialized
- Project documentation started

## Technology Stack

- Python
- Git
- VS Code
- Virtual Environment (venv)

## Project Goal

Build an AI agent capable of:
- Selecting a technology topic
- Researching the topic
- Generating LinkedIn posts
- Maintaining posting history
- Supporting future AI agents
